# millipede-cpp-z
:bug: Print a beautiful millipede-z
## Installation



Using no Windows

````sh
$ g++ -W -wall -error milipede.cpp -o millipede
```

## Usage

````sh
millipede [PARAMS]...

millipede <size>
millipede <size> <message>
```

## Examples


````sh
$ ./millipede
(default size iz 42)

$ ./millipede 10

$ ./millipede 15 MILLIPEDE
```
